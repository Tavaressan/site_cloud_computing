Amazon Aurora

Banco de dados NoSQL orientado a documentos compatível com MongoDB. Armazena dados em formato JSON para aplicações modernas.

Amazon DynamoDB

Banco NoSQL chave-valor extremamente rápido e escalável. Serviço serverless muito usado em apps mobile, jogos e sistemas com milhões de usuários.

Ferramenta para controlar serviços da AWS pelo terminal permitindo criar recursos, fazer upload para S3 e automatizar tarefas.

Ferramenta de automação de deploy que cria pipelines para automatizar todo o fluxo de entrega de software.
