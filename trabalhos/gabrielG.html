S3

Amazon S3

Buckets para armazenamento de objetos (arquivos) tipo fotos, logs, vídeos, backups. Super escalável. "Clientes de todos os tamanhos e setores podem usar o Amazon S3 para armazenar e proteger qualquer volume de dados para uma variedade de casos de uso, como data lakes, sites, aplicações móveis, backup e restauração, arquivamento, aplicações corporativas, dispositivos IoT e análises de big data."

Armazenamento Escalável

Lambda

AWS Lambda

O Lambda é um serviço de computação orientado por eventos, ele usa um paradigma de programação diferente das aplicações Web tradicionais, utilizando Computação serverless(um modelo de execução de aplicações na nuvem onde o provedor gerencia a infraestrutura, permitindo que os desenvolvedores se concentrem no código, sem se preocupar com a gestão de servidores) — você escreve uma função, define um gatilho, e a AWS executa sem precisar de servidor fixo.

Event-Driven Serverless

CloudFront

Amazon CloudFront

O CloudFront é um serviço de entrega de conteúdo (CDN) da AWS que distribui conteúdo para os usuários com baixa latência e alta velocidade. Ele armazena cópias do conteúdo em servidores distribuídos globalmente, permitindo que os usuários acessem o conteúdo de forma rápida e eficiente, independentemente da localização geográfica.

CDN Entrega de Conteúdo

EC2

Amazon Elastic Compute Cloud

O EC2 é um serviço de computação em nuvem que oferece capacidade de computação redimensionável na nuvem. Ele permite que os usuários criem e gerenciem máquinas virtuais, conhecidas como instâncias, para executar aplicativos e serviços. O EC2 é altamente flexível, permitindo que os usuários escolham entre uma variedade de tipos de instâncias, sistemas operacionais e configurações de rede para atender às suas necessidades específicas.

Máquinas Virtuais Redimensionável

VPC

Amazon Virtual Private Cloud

O VPC é um serviço da AWS que permite criar uma rede virtual isolada na nuvem. Ele oferece controle sobre o espaço de endereçamento IP, subnets, gateways e segurança, permitindo que os usuários criem ambientes de rede personalizados para seus aplicativos e serviços.

Segurança Isolamento

RDS

Amazon Relational Database Service

O Amazon RDS é um serviço gerenciado de banco de dados relacional da AWS. Ele permite criar, operar e escalar bancos de dados na nuvem sem precisar gerenciar a infraestrutura manualmente. O serviço automatiza tarefas como backups, atualizações, replicação e monitoramento, facilitando o gerenciamento de bancos como MySQL, PostgreSQL, MariaDB, Oracle e SQL Server.

Banco de Dados Gerenciado

IAM

AWS Identity and Access Management

O IAM é o serviço de gerenciamento de identidades e permissões da AWS. Ele permite controlar quem pode acessar recursos da nuvem e quais ações podem ser executadas. Com ele é possível criar usuários, grupos, funções e políticas de acesso, garantindo segurança e controle sobre os serviços da AWS utilizados em uma organização.

Segurança Controle de Acesso

Route 53

Amazon Route 53

O Route 53 é o serviço de DNS da AWS responsável por direcionar o tráfego da internet para aplicações hospedadas na nuvem. Ele traduz nomes de domínio em endereços IP e pode ser usado para registrar domínios, configurar roteamento inteligente e melhorar a disponibilidade das aplicações.

DNS Roteamento

CloudWatch

Amazon CloudWatch

O CloudWatch é o serviço de monitoramento da AWS que coleta métricas, logs e eventos de aplicações e infraestrutura. Ele permite analisar performance, criar alertas automáticos e dashboards, facilitando a observabilidade, diagnóstico de problemas e otimização de recursos na nuvem.

Monitoramento Observabilidade
