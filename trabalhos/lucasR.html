AWS IAM

AWS Identity and Access Management

Base de toda segurança na AWS. Permite criar e gerenciar usuários, grupos, funções e políticas de acesso. Controla quem pode fazer o quê em quais recursos da conta AWS com precisão granular.

Identidade Acesso Políticas

Amazon Cognito

Autenticação para Aplicações

Gerencia autenticação e autorização de usuários em aplicações web e mobile. Oferece cadastro, login, recuperação de senha e federação com provedores como Google, Facebook e SAML.

Autenticação OAuth Federação

AWS KMS

AWS Key Management Service

Criação e gerenciamento centralizado de chaves de criptografia. Integrado a praticamente todos os serviços AWS (S3, RDS, EBS), garante que dados em repouso e em trânsito sejam protegidos com criptografia gerenciada.

Criptografia Chaves CMK Conformidade

AWS ACM

AWS Certificate Manager

Provisiona, gerencia e renova automaticamente certificados SSL/TLS para proteger comunicações HTTPS. Integrado ao CloudFront, ALB e API Gateway, elimina a complexidade de renovação manual de certificados.

SSL/TLS HTTPS Certificados

Secrets Manager

AWS Secrets Manager

Armazena e rotaciona automaticamente segredos como senhas de banco de dados, tokens de API e chaves de acesso. Elimina credenciais hardcoded no código e garante acesso seguro e auditável aos segredos.

Segredos Rotação Credenciais

Amazon GuardDuty

Detecção Inteligente de Ameaças

Monitora continuamente logs de CloudTrail, VPC Flow e DNS para detectar comportamentos maliciosos e atividades suspeitas usando machine learning. Identifica ameaças como mineração de criptomoedas, acesso incomum e exfiltração de dados.

Detecção Machine Learning Ameaças

AWS WAF

AWS Web Application Firewall

Firewall para aplicações web que bloqueia ataques como SQL Injection, XSS e requisições maliciosas. Integra-se ao CloudFront, ALB e API Gateway, com regras gerenciadas pela AWS e customizáveis pelo usuário.

Firewall SQL Injection XSS

AWS Shield

Proteção contra Ataques DDoS

Protege aplicações contra ataques DDoS. O Shield Standard é gratuito e ativo automaticamente para todos os clientes AWS. O Shield Advanced oferece proteção gerenciada 24/7, suporte especializado e reembolso de custos em caso de ataque.

DDoS Standard Advanced

Amazon Inspector

Análise Automática de Vulnerabilidades

Analisa automaticamente vulnerabilidades de segurança em instâncias EC2, imagens de container (ECR) e funções Lambda. Gera relatórios priorizados por severidade e se integra ao Security Hub para visão centralizada dos riscos.

Vulnerabilidades EC2 / Lambda CVE
