Amazon GameLift

Hosting de jogos

Serviço gerenciado para implantar e operar servidores de jogos multiplayer. Ele otimiza a latência ao aproximar os servidores dos jogadores e ajusta automaticamente a capacidade para reduzir custos em momentos de baixa demanda.

Cloud Jogos

AWS Lambda

Sem Servidor

Permite a execução de código em resposta a eventos. Ideal para arquiteturas de microsserviços, o serviço escala automaticamente e cobra apenas pelo tempo de processamento.

Hook Evento

Amazon S3

Serviço de Armazenamento

Uma plataforma de armazenamento de objetos. É muito utilizado para backup de dados, hospedagem de sites estáticos e como repositório principal para Data Lakes, permitindo o acesso seguro de qualquer lugar na web.

Armazenamento Cloud

Amazon DynamoDB

Serviço de Banco NoSQL

Banco de dados NoSQL de chave-valor e documentos que oferece desempenho consistente. É um serviço totalmente gerenciado, multi-região e com segurança integrada, ideal para aplicações móveis, web e jogos.

Armazenamento NoSQL

Amazon Route 53

Serviço DNS escalável

Serviço de DNS disponível e escalável que conecta as solicitações dos usuários a infraestruturas executadas na AWS ou fora dela. Ele faz verificações de integridade para rotear o tráfego apenas para endpoints seguros e suporta registros de nomes de domínio.

Rede DNS

Amazon VPC

Nuvem virtual privada

Proporciona uma rede virtual logicamente isolada para testar recursos AWS em um ambiente seguro e controlado. O usuário define o intervalo de endereços IP, sub-redes, tabelas de roteamento e gateways de rede, garantindo isolamento total de outros clientes.

Rede Segurança

Amazon CloudFront

Rede de envio de conteúdo

Serviço de CDN que acelera a distribuição de conteúdos estáticos e dinâmicos para usuários finais em todo o mundo. Ele utiliza uma rede global de Edge Locations para entregar dados com baixa latência.

Rede Distribuição

AWS IAM

Controle de acesso

Controla o acesso de usuários e grupos aos recursos da nuvem. Através do IAM, é possível criar políticas de segurança específicas e gerenciar credenciais, garantindo conformidade e segurança em toda a conta.

Segurança Cloud

Amazon RDS

Serviço de banco SQL

Serviço gerenciado que facilita a configuração, operação e escalabilidade de bancos de dados relacionais. Ele automatiza tarefas demoradas como backups, atualizações de software e provisionamento de hardware, garantindo alta disponibilidade.
